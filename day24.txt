# Create headings for the table; where we created the structure for the Database
-> There are 4 operations: Create, Read, Update, Delete
-> Now, Insert DATA in it using "Insert" in it.
Syntax:
INSERT INTO `ExchangeRate` (`id`, `date`, `iso3`, `name`, `unit`, `buy`, `sell`) VALUES (NULL, '2026-04-01', 'NEP', 'Nepal', '1', '150', '150');

# To read the table:
SELECT * FROM `ExchangeRate`

Also, SELECT * FROM `ExchangeRate` where id = 1
Also, SELECT * FROM `ExchangeRate` limit = 2
-> limit: shows the number of rows from the table; like "2" shows 2 rows of data,


# Then, to update the table: 
We edit it from pressing the "edit" in the row of the table that we want to update; 
Syntax:
UPDATE `ExchangeRate` SET `buy` = '159', `sell` = '180' WHERE `ExchangeRate`.`id` = 1;

# In Update table, if 'WHERE' command is not mentioned, 
then whole table changes and it cannot be undone afterwards.

# Delete the row: ; the "Delete" command is given from the row.

Syntax:
DELETE FROM ExchangeRate WHERE `ExchangeRate`.`id` = 3;
or, 
DELETE FROM ExchangeRate WHERE id = 3;

# In order to delete the unique id = 3;

If "WHERE" clause in the Delete operation is not included, then it deletes the whole rows in the table.
Syntax: DELETE FROM ExchangeRate;
